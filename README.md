# Canvas Lock

A tiny quality-of-life add-on for Blender: adds an **aspect ratio lock** and an **orientation swap** button directly inside Blender's native Format panel (Properties > Output).

With the lock on, dragging Resolution X automatically adjusts Resolution Y (and vice versa) to keep your current aspect ratio. The swap button flips portrait/landscape in one click.

No new panels, no configuration. Install it and the lock appears where you'd expect it.

## Install

Blender 4.2+: Edit > Preferences > Get Extensions > search "Canvas Lock" (or Install from Disk with the release ZIP).
